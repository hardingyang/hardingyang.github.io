医疗信息接诊模块

接诊模块设计

数据库表：

Patient表

列名 数据类型 是否可空 备注
patient_id int no 患者主键id
name varchar(20) no 患者姓名
age short no 患者年纪
phone char(11) no 患者手机
address varchar(100) no 患者住址
medical_history bool no 患者病史
medical_info varchar(300) no 患者患病信息

doctor表：

列名 数据类型 是否可空 备注
doctor_id int no 医生主键id
name varchar(20) no 医生姓名
age short no 医生年纪
phone char(11) no 医生手机
address varchar(100) no 医生住址
degree int no 医生级别

diagnose表：

列名 数据类型 是否可空 备注
doctor_id int no 医生主键id
patient_id int no 患者主键id
diagnose_time datetime no 诊断时间
diagnose_info varchar(200) no 诊断结果信息

hospital_bed表:

列名 数据类型 是否可空 备注
bed_id int no 病床主键id
bed_is_null bool no 病床是否有人住

bed_patient表：

列名 数据类型 是否可空 备注
bed_id int no 病床主键id
patient_id int no 病人主键id

items表：

列名 数据类型 是否可空 备注
item_name varchar(50) no 收费项目名字
item_price Decimal no 项目价格
item_info varchar(100) no 项目信息说明

medicine表：

列名 数据类型 是否可空 备注
medicine_name varchar(50) no 药物名称
medicine_price decimal no 药物价格
medicine_number int no 药物剩余个数
